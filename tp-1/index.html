Blender

Blender es un software de gráficos 3D multiplataforma, dedicado especialmente al modelado, iluminación, renderizado, la animación y creación de gráficos 3D. Es usado para crear animaciones, efectos visuales, modelos para impresión 3D, y assets para videojuegos.

En sus orígenes en 1994, Blender era distribuido de forma gratuita pero sin el código fuente, hasta que en 2002 pasó a ser software libre. Actualmente es compatible con todas las versiones de Windows, macOS, GNU/Linux, Android, Solaris, FreeBSD e IRIX.

Blender fue utilizado para crear la película Flow (2024), ganadora del premio Óscar.

Características

  • Multiplataforma, libre, gratuito y con un tamaño de origen realmente pequeño comparado con otros paquetes de 3D, dependiendo del sistema operativo en el que se ejecuta.
  • Capacidad para una gran variedad de primitivas geométricas, incluyendo curvas, mallas poligonales, vacíos, NURBS, metaballs.
  • Junto a las herramientas de animación se incluyen cinemática inversa, deformaciones por armadura o cuadrícula, vértices de carga y partículas estáticas y dinámicas.
  • Edición de audio y edición de vídeo.
  • Características interactivas para juegos como detección de colisiones, recreaciones dinámicas y lógica.
  • Posibilidades de renderizado interno versátil e integración externa con potentes trazadores de rayos o "raytracer" libres como kerkythea, YafRay o Yafrid.
  • Lenguaje Python para automatizar o controlar varias tareas.
  • Blender acepta formatos gráficos como PNG, JPEG, TIFF, WEBP, entre otros. También puede leer ficheros OBJ, FBX, PLY, STL, GLTF, entre otros.
  • Simulaciones físicas.
  • Modificadores apilables, para la aplicación de transformación no destructiva sobre mallas.
  • Capacidad para hacer captura de movimiento.
